# Age Calculator

A responsive Age Calculator built with HTML, CSS, and JavaScript. Users can enter their date of birth to instantly calculate their current age. The application validates user input and accurately determines whether the user's birthday has occurred this year before displaying their age.

## Features

- Enter your date of birth using a date picker
- Calculates your current age instantly
- Validates user input before calculation
- Correctly handles birthdays that haven't occurred yet this year
- Displays a friendly age message
- Responsive and clean user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Date Object
- DOM Manipulation
- Event Listeners
- Conditional Statements
- Input Validation
